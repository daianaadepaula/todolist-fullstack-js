### Run Locally

To run **ToDoList** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/daianaadepaula/todolist-fullstack-js.git
```

Windows:

```bash
git clone https://github.com/daianaadepaula/todolist-fullstack-js.git
```
